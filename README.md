# Brow Studio Landing Page

Одностраничный сайт-витрина для броу-студии с блоками услуг, преимуществ, процесса записи и контактной формой обратной связи.

## Структура
- `index.html` — разметка страницы с секциями геро, услуги, безопасность и форма.
- `styles.css` — стилизация тёмной темы, сеток карточек и адаптивных отступов.

## Просмотр
Откройте `index.html` в браузере или поднимите простой сервер:

```bash
python3 -m http.server 8000
```

Страница будет доступна по адресу http://localhost:8000.
